Welcome to Project Newcastle

Getting Started

Project Newcastle uses the openshift service for prototyping and development. The project sources are available in github. You can easily set up your own openshift application environment and build and deploy updates to your project newcastle instance.

Using the Demo

Project Newcastle is currently in the very early stages of development, so there is only limited functionality. As more functionality is added, additional links should be places on this welcome page for easy navigation. You can see a list of the current database tables here.
